User Manual

Welcome to the Help Page. Here are the key features and instructions for using the ADHD Task Manager:

Getting Started

To get started with the app, follow these steps:

  1. Open the app on your Android device.
  2. Sign in with your Google account or anonymously.
  3. Explore the app and enjoy its features!

Adding A New Task

This allows you to create a new to do task:

  1. Tap on the "Plus" button on the bottom right.
  2. Enter the required information in the provided fields.
  3. Task name and description is compulsory information
  4. Fill in the optional data fields.
  5. Tap the add button

Editing an Existing Task

Use this to modify existing task details. Follow these steps:

  1. Click on the edit symbol.
  2. A pop up screen appears.
  3. Make changes as desired..
  4. Tap the "Add" button to apply the changes.

View the Leaderboard

Interested in knowing the scores!. Follow these steps:

  1. Navigate to the "Home Page" section in the app.
  2. Select leaderboard icon.
  3. View the scores

Using Feature B

Feature B is a powerful tool to accomplish ABC. Follow these steps:

  1. Navigate to the "Feature B" section in the app.
  2. Select the desired options from the list.
  3. Use the slider to adjust the settings.
  4. Tap the "Apply" button to apply the changes.

Troubleshooting

If you encounter any issues while using the app, try the following:

  • Check your internet connection.
  • Ensure that you have the latest version of the app installed.
  • Restart your device and relaunch the app.
  • Contact our support team at support@example.com for further assistance.
